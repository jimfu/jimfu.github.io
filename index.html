GOAWAY

The interfaces
Actually I don't code this app, I just set the flow path and the interfaces.
It is an Android app,so the most difficult part of this project is that I use iPhone, so you may see a lot of iOS design language here. Eventually, after many discussions and feedbacks from users, it comes with favourable responses.
Wireframe
My first try of wireframe, I write down all the flow and connect them to each other in detail. It is the hardest part in UX at the beginning because you have to make sure the flow has no conflict and has to be clear on every steps.
